# ASSTF Launch Copy Assets

This folder contains ready-to-use copy for the Stage 1 public launch.

## Files

- `blog_post.md` — Company blog / Medium / dev.to
- `show_hn.md` — Hacker News Show HN submission
- `reddit_machinelearning.md` — r/MachineLearning post
- `x_thread.md` — Twitter/X thread
- `linkedin.md` — LinkedIn founder/company post
- `chinese.md` — 知乎 / 机器之心 / 掘金 Chinese version

## Launch Timing

- **Best days:** Tuesday or Wednesday
- **Best time:** 9:00–11:00 AM US Eastern Time
- Sequence: arXiv/blog teaser (T-1) → GitHub Release + X/LinkedIn (T-Day AM) → Show HN (T-Day AM) → Reddit (T+1) → KOL amplification (T+2)
