# ASSTF Documentation

Welcome to the official documentation for **ASSTF: Adaptive State-Space Transfer Function**.

ASSTF is a PyTorch framework for parameter-efficient, inference-adaptable neural layers — designed for edge AI, continual learning, and on-device personalization.
